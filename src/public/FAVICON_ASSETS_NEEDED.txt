NOTINOW - FAVICON ASSETS NEEDED
================================================================================

⚠️  IMPORTANT: Binary PNG/ICO files cannot be auto-generated in this environment.
    You need to manually add the following files to the /public folder.

================================================================================
REQUIRED FILES:
================================================================================

1. favicon.ico
   - Size: Multi-resolution ICO (16x16, 32x32, 48x48)
   - Format: ICO
   - Location: /public/favicon.ico

2. favicon-16x16.png
   - Size: 16x16 pixels
   - Format: PNG (transparent background)
   - Location: /public/favicon-16x16.png

3. favicon-32x32.png
   - Size: 32x32 pixels
   - Format: PNG (transparent background)
   - Location: /public/favicon-32x32.png

4. apple-touch-icon.png
   - Size: 180x180 pixels
   - Format: PNG (can have background)
   - Location: /public/apple-touch-icon.png

5. icon-192.png
   - Size: 192x192 pixels
   - Format: PNG (transparent background)
   - Location: /public/icon-192.png

6. icon-512.png
   - Size: 512x512 pixels
   - Format: PNG (transparent background)
   - Location: /public/icon-512.png

================================================================================
DESIGN SPECIFICATIONS:
================================================================================

Logo: Lowercase "n" character
Color: #7C3AED (violet/purple) - Notinow primary brand color
Background: White or transparent
Typography: Clean, modern sans-serif (system-ui)
Border Radius: 6px for backgrounds

================================================================================
QUICK GENERATION TOOLS:
================================================================================

Option 1 - Online (Easiest):
→ https://favicon.io/favicon-converter/
→ Upload the blue "n" logo image
→ Download and extract to /public folder

Option 2 - RealFaviconGenerator (Most comprehensive):
→ https://realfavicongenerator.net/
→ Upload the logo
→ Customize settings
→ Download package and extract to /public folder

Option 3 - Figma (Manual):
→ Open the provided logo in Figma
→ Export at each required size as PNG
→ Use an ICO converter for favicon.ico
